# タスク管理アプリ 要件定義書

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| 目的 | スクール課題として学習用に作成するTrello風タスク管理アプリ |
| 利用者 | 個人（自分のみ） |
| 形態 | フロントエンドのみ（バックエンド・認証不要） |

---

## 画面構成

ボード画面 **1画面のみ**。カラムは「未着手」「作業中」「完了」の固定3列。

```
┌──────────────────────────────────────────────────────┐
│  タスク管理ボード                                      │
├──────────────────┬──────────────────┬────────────────┤
│     未着手        │     作業中        │     完了        │
│                  │                  │                │
│ ┌──────────────┐ │ ┌──────────────┐ │ ┌────────────┐ │
│ │ タスクA      │ │ │ タスクC      │ │ │ タスクE    │ │
│ │ 優先度: 高   │ │ │ 優先度: 中   │ │ │            │ │
│ │ 期限: 4/30  │ │ │              │ │ │            │ │
│ │   [編集][削除]│ │ │  [編集][削除]│ │ │[編集][削除]│ │
│ └──────────────┘ │ └──────────────┘ │ └────────────┘ │
│                  │                  │                │
│  [+ カード追加]  │  [+ カード追加]  │ [+ カード追加] │
└──────────────────┴──────────────────┴────────────────┘
```

---

## 機能一覧

### Must（必須機能）

| # | 機能 | 詳細 |
|---|------|------|
| M1 | カラム表示 | 「未着手」「作業中」「完了」の固定3カラムを横並びで表示する |
| M2 | カード作成 | タイトル（必須）を入力してカードを追加する |
| M3 | カード編集 | タイトル・説明文・優先度・期限を編集できる |
| M4 | カード削除 | カードを削除する（確認なし or 確認ダイアログ） |
| M5 | カラム間移動 | ドラッグ&ドロップでカードを別カラムへ移動する |
| M6 | データ永続化 | localStorageに保存し、ブラウザを閉じても消えない |

### Want（あると嬉しい機能）

| # | 機能 | 詳細 |
|---|------|------|
| W1 | 期限切れ警告 | 期限を過ぎたカードを赤くハイライトして視覚的に警告する |
| W2 | カラム内並び替え | カラム内でカードの順序をドラッグ&ドロップで変更できる |
| W3 | カード枚数バッジ | 各カラムのヘッダーにカード枚数を表示する |
| W4 | 優先度フィルター | 優先度でカードを絞り込んで表示できる |

---

## カードのデータ項目

| 項目 | 必須 | 型 | 備考 |
|------|------|----|------|
| id | ○ | string | 自動生成（crypto.randomUUID等） |
| title | ○ | string | カードのタイトル |
| description | - | string | 説明文（任意） |
| priority | - | `"high"` \| `"medium"` \| `"low"` | 優先度（任意） |
| dueDate | - | string (YYYY-MM-DD) | 期限（任意） |
| createdAt | ○ | string (ISO 8601) | 作成日時（自動） |

---

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | React（Vite） |
| スタイリング | Tailwind CSS |
| ドラッグ&ドロップ | dnd-kit |
| 状態管理 | useState / useReducer |
| データ永続化 | localStorage |

---

## スコープ外

- ユーザー認証・複数ユーザー対応
- サーバーサイド / データベース連携
- カラムの追加・削除・名前変更
- ラベル・タグ機能
- 通知・リマインダー
- スマートフォン対応（デスクトップ優先）
