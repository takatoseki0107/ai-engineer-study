## APIエンドポイント

| メソッド | パス | 内容 |
|---------|------|------|
| GET | /api/tasks | 全タスク取得 |
| POST | /api/tasks | タスク作成 |
| PATCH | /api/tasks/:id | タスク更新（内容・ステータス・並び順） |
| DELETE | /api/tasks/:id | タスク削除 |
